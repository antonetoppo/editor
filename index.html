Title

Enter Text!!...

Features

1) Pressing "Enter" key on any para would create a new para and set focus on new para.
2) Double Click a word or select using "SHIFT" key and the arrow keys would present a tooltip.
3) Tooltip gives the ability to change the style of selected characters.
4) Style Options are to Bold(B), Underline(U) or to Red(R).
5) Style Options can be toggled on or off.
6) One set multiplestyle on same selecter characters.
7) In order to re-arrange the paragraphs toggle it on or off via button on top right of the editor.
8) Paragraphs are non-editable when in arrangement mode.
9) Anchor tag can generate link when the "DONE" button is pressed.
9) Word of length 4 ending with 'space' or '.' or ',' or 'end of para i.e. $' will be replaced with random word on pressing the "Randomize" Button.
